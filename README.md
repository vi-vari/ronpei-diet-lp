# 体質改善ダイエット講座 LP

Figma デザイン（[LP / node 428:3](https://www.figma.com/design/Nd86bU4YvLSdwvI1IjcYkD/LP?node-id=428-3)）を
もとに実装した、レスポンシブ対応のランディングページです。

## 構成

```
.
├── index.html        # ページ本体（1カラム LP）
├── css/style.css     # スタイル（デザイントークン + レスポンシブ）
└── js/script.js      # LINE リンク一括設定 + スティッキー CTA 制御
```

## レスポンシブ

- モバイルファースト。コンテンツは最大 `700px` 幅で中央寄せ。
- 画像は `width:100%` + `aspect-ratio` / `object-fit:cover` で可変。
- 見出し・余白は `clamp()` でビューポートに応じて調整。
- 768px 未満では画面下部にスティッキー LINE CTA を表示。
- `prefers-reduced-motion` に対応。

## 設定が必要な項目

- **LINE URL**: `js/script.js` の `LINE_URL` に実際の LINE 追加 URL を設定してください。
  設定すると全 CTA ボタン（固定 CTA を含む）に反映されます。

## ⚠️ 画像について（重要）

現在、各画像は **Figma の CDN URL**（`https://www.figma.com/api/mcp/asset/...`）を
直接参照しています。**これらの URL は発行から約7日で失効します。**

本番運用の際は画像をローカル（例: `images/`）へ保存し、`index.html` 内の
`src` を相対パスへ差し替えてください。差し替え対象は以下の通りです。

| セクション | 用途 |
| --- | --- |
| FV | ヒーロー画像 1点 |
| ビフォーアフター | 実績画像 7点 |
| お客様の声 | 投稿写真 12点 |
| ３つのこだわり | 施術・空間写真 3点 |
| スタッフ紹介 | 代表の顔写真 1点 |
| 店舗情報 | 店舗写真 1点 / 特典（書籍）画像 1点 |
| 成功する理由 | 各理由の写真 3点 + 柔道整復師免許 1点 |
