## レスポンシブ

- モバイルファースト。コンテンツは最大 `700px` 幅で中央寄せ。
- 画像は `width:100%` + `aspect-ratio` / `object-fit:cover` で可変。
- 見出し・余白は `clamp()` でビューポートに応じて調整。
- 768px 未満では画面下部にスティッキー LINE CTA を表示。
- `prefers-reduced-motion` に対応。

## 設定が必要な項目

- **LINE URL**: `js/script.js` の `LINE_URL` に実際の LINE 追加 URL を設定してください。
  設定すると全 CTA ボタン（固定 CTA を含む）に反映されます。

## 📷 画像一覧（`images/` フォルダに配置してください）

`index.html` は下表のファイル名で `images/` 配下の画像を参照します。
**Canva の元データから、この推奨ファイル名・サイズで書き出して `images/` に置いてください。**
（ファイル名を変える場合は `index.html` の `src` も合わせて変更してください）

### 全体の書き出し方針

- **推奨サイズ = 表示幅の約2倍**（Retina／高解像度ディスプレイ対応）。表内の値で書き出せば OK です。
- **形式**: 写真は `JPG`（画質 80〜85%）。文字が入るグラフィック（FV・ビフォーアフター・特典）は
  文字がぼやけないよう `PNG` でも可（その場合は拡張子を `.png` に変え、`index.html` の `src` も変更）。
- **軽量化**: 各ファイル 200〜400KB 程度を目安に圧縮すると表示が速くなります。
- **比率厳守**: 下表の比率で書き出すと、レイアウトのトリミングずれがありません。
